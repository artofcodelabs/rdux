# Rudux
Short description and motivation.

## Usage

```bash
$ bin/rails rudux:install:migrations
```

## Installation
Add this line to your application's Gemfile:

```ruby
gem 'rudux'
```

And then execute:
```bash
$ bundle
```

Or install it yourself as:
```bash
$ gem install rudux
```

## Test

### Setup

```bash
$ cd test/dummy
$ bin/rails db:migrate RAILS_ENV=test
```

# Run (from the top directory)

```bash
$ bin/test
```
