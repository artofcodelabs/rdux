# Rdux
Short description and motivation.

## Usage

```bash
$ bin/rails rdux:install:migrations
```

## Installation
Add this line to your application's Gemfile:

```ruby
gem 'rdux'
```

And then execute:
```bash
$ bundle
```

Or install it yourself as:
```bash
$ gem install rdux
```

## Test

```bash
$ DB=postgres bin/test
$ DB=sqlite bin/test
```
